(static) cssInjector(cssRules, IDopt, referenceElementopt)

Insert base stylesheet into DOM

Creates a new <style>...</style> element from the named text string(s) and inserts it.

Parameters:
Name Type Attributes Description
cssRules string | Array.<string>
ID string <optional>
referenceElement string | Element | undefined | null <optional>
  • undefined type (or omitted): injects stylesheet at top of <head>...</head> element
  • null value: injects stylesheet at bottom of <head>...</head> element
  • Element type: injects stylesheet immediately before given element, wherever it is found.
  • string type: injects stylesheet immediately before given first element found that matches the given css selector.
Returns:

A reference to the newly created <style>...</style> element.
